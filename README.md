1. Your First Express Server

2. Handling Routes with Express
   - 웹에는 get, post 두가지 요청 방식
     : get = 단순히 정보를 보여줌
     : post = 정보를 전달해줌
   - 요청시 request, response 두가지의 파라메터가 있다
     : request = 웹에서의 전달하는 정보가 들어있다
     : response = 웹에서의 요청을 대답한다.
